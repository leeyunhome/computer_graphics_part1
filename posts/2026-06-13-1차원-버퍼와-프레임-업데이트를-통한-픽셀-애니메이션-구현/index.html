    1м°Ёмӣҗ лІ„нҚјмҷҖ н”„л Ҳмһ„ м—…лҚ°мқҙнҠёлҘј нҶөн•ң н”Ҫм…Җ м• лӢҲл©”мқҙм…ҳ кө¬нҳ„

    н•ҷмҠөмһҗлҠ” мқҙлІҲ м»Өл°Ӣмқ„ нҶөн•ҙ CPU мёЎм—җм„ң м •мқҳлҗң 2м°Ёмӣҗ лһҳмҠӨн„°(Raster) к·ёлһҳн”ҪмҠӨ лҚ°мқҙн„°лҘј 1м°Ёмӣҗ м—°мҶҚ л©”лӘЁлҰ¬ лІ„нҚјлЎң кҙҖлҰ¬н•ҳкі , л§Ө н”„л Ҳмһ„л§ҲлӢӨ мғҒнғңлҘј к°ұмӢ н•ҳм—¬ нҷ”л©ҙм—җ мӣҖм§ҒмқҙлҠ” м• лӢҲл©”мқҙм…ҳмқ„ н‘ңнҳ„н•ҳлҠ” н•өмӢ¬ к·ёлһҳн”ҪмҠӨ кё°мҙҲ к°ңл…җмқ„ н•ҷмҠөн–ҲмҠөлӢҲлӢӨ.

    1. н•ҷмҠөмһҗк°Җ м§Ғм ‘ кө¬нҳ„н•ҳл©° мқҙн•ҙн•ң н•өмӢ¬ к°ңл…җ

    (1) 2м°Ёмӣҗ кіөк°„мқҳ 1м°Ёмӣҗ м„ нҳ• л©”лӘЁлҰ¬ л§Өн•‘

    м»ҙн“Ён„° к·ёлһҳн”ҪмҠӨм—җм„ң нҷ”л©ҙ(Canvas)мқҖ 2м°Ёмӣҗ к·ёлҰ¬л“ң(\(Width \times Height\)) нҳ•нғңлҘј к°Җм§‘лӢҲлӢӨ. н•ҳм§Җл§Ң л№„л””мҳӨ л©”лӘЁлҰ¬(VRAM)лӮҳ мӢңмҠӨн…ң л©”лӘЁлҰ¬(RAM)лҠ” кё°ліём ҒмңјлЎң 1м°Ёмӣҗ л°°м—ҙ кө¬мЎ°мһ…лӢҲлӢӨ. н•ҷмҠөмһҗлҠ” мқҙлҘј л§Өн•‘н•ҳлҠ” кіөмӢқмқ„ мқҙн•ҙн•ҳкі  м Ғмҡ©н–ҲмҠөлӢҲлӢӨ.

    2м°Ёмӣҗ мўҢн‘ң \((x, y)\)лҘј 1м°Ёмӣҗ мқёлҚұмҠӨ \(index\)лЎң ліҖнҷҳн•ҳлҠ” кіөмӢқмқҖ лӢӨмқҢкіј к°ҷмҠөлӢҲлӢӨ. $\(index = x + (y \times Width)\)$

    л°ҳлҢҖлЎң, 1м°Ёмӣҗ мқёлҚұмҠӨ \(i\)лЎңл¶Җн„° нҳ„мһ¬ н”Ҫм…Җмқҳ 2м°Ёмӣҗ мўҢн‘ң \((x, y)\)лҘј м—ӯмӮ°н•ҳл©ҙ лӢӨмқҢкіј к°ҷмҠөлӢҲлӢӨ. $\(x = i \pmod{Width}\)$ $\(y = \lfloor \frac{i}{Width} \rfloor\)$

    н•ҷмҠөмһҗлҠ” мқёлҚұмҠӨ \(i\)лҘј л§Ө н”„л Ҳмһ„ мҲңм°Ём ҒмңјлЎң мҰқк°ҖмӢңнӮҙмңјлЎңмҚЁ, л№Ёк°„мғү н”Ҫм…Җмқҙ мўҢмёЎ мғҒлӢЁ \((0, 0)\)м—җм„ң мӢңмһ‘н•ҳм—¬ мҡ°мёЎмңјлЎң мқҙлҸҷн•ҳлӢӨк°Җ нҷ”л©ҙ лҒқм—җ лҸ„лӢ¬н•ҳл©ҙ м•„лһҳ н–үмқҳ мІ« лІҲм§ё м—ҙлЎң мӨ„л°”кҝҲ(Line-wrap)н•ҳл©° мқҙлҸҷн•ҳлҠ” мЈјмӮ¬(Scanning) л°©мӢқмқ„ мҪ”л“ңлЎң кө¬нҳ„н•ҙ лғҲмҠөлӢҲлӢӨ.

    (2) м•Ҳм „м„ұ л°Ҹ л°©м–ҙм Ғ н”„лЎңк·ёлһҳл°Қ (Type Safety & Defensive Programming)

    мқҙлІҲ ліҖкІҪмӮ¬н•ӯмқҳ н•өмӢ¬мқҖ л¶Җнҳё мһҲлҠ” м •мҲҳнҳ•(int)м—җм„ң л¶Җнҳё м—ҶлҠ” м •мҲҳнҳ•(size_t)мңјлЎңмқҳ нғҖмһ… ліҖнҷҳкіј кІҪкі„ кІҖмӮ¬ мЎ°кұҙмӢқмқҳ к°ңм„ мһ…лӢҲлӢӨ.

    • нғҖмһ… м•Ҳм •м„ұ (size_t): std::vector::size()лҠ” м»Ён…Ңмқҙл„Ҳмқҳ нҒ¬кё°лҘј лӮҳнғҖлӮҙлҠ” л¶Җнҳё м—ҶлҠ” м •мҲҳнҳ•(size_t)мқ„ л°ҳнҷҳн•©лӢҲлӢӨ. мқҙм „ мҪ”л“ңмқҳ int iмҷҖ л№„көҗн•  кІҪмҡ° м•”л¬өм Ғ нҳ•ліҖнҷҳ л°Ҹ м„ңлӘ… л№„көҗ кІҪкі (Signed/Unsigned mismatch warning)к°Җ л°ңмғқн•  мҲҳ мһҲмҠөлӢҲлӢӨ. мқҙлҘј size_tлЎң нҶөмқјн•ҳм—¬ нғҖмһ… м•Ҳм „м„ұмқ„ лҶ’мҳҖмҠөлӢҲлӢӨ.
    • л°©м–ҙм Ғ кІҪкі„ мЎ°кұҙ (>=): кё°мЎҙмқҳ i == pixels.size() - 1 мЎ°кұҙмқҖ мҷ„лІҪнһҲ мҲңм°Ём Ғмқё нқҗлҰ„м—җм„ңлҠ” мһ‘лҸҷн•ҳм§Җл§Ң, л©ҖнӢ°мҠӨл Ҳл”©мқҙлӮҳ н”„л Ҳмһ„ л“ңлЎӯ, нҳ№мқҖ мҲҳм№ҳ мҳӨлІ„н”ҢлЎңмҡ° л“ұмқҳ мҳҲмҷё мғҒнҷ©м—җм„ң iк°Җ кІҪкі„к°’мқ„ мҙҲкіјн•ҙ лІ„лҰҙ кІҪмҡ° л¬ҙн•ң лЈЁн”„лӮҳ мң нҡЁн•ҳм§Җ м•ҠмқҖ л©”лӘЁлҰ¬ м°ёмЎ°(Out of Bounds) л¬ём ңлҘј л°ңмғқмӢңнӮ¬ мҲҳ мһҲмҠөлӢҲлӢӨ. мқҙлҘј >= л№„көҗ м—°мӮ°мңјлЎң ліҖкІҪн•ЁмңјлЎңмҚЁ лІ„нҚј мҳӨлІ„н”ҢлЎңмҡ°лҘј мӣҗмІң м°ЁлӢЁн•ҳлҠ” л°©м–ҙм Ғ н”„лЎңк·ёлһҳл°Қмқ„ мӢӨмІңн–ҲмҠөлӢҲлӢӨ.

    2. н”„л Ҳмһ„ м• лӢҲл©”мқҙм…ҳ м•Ңкі лҰ¬мҰҳ нқҗлҰ„ (мқҳмӮ¬мҪ”л“ң)

    л§Ө н”„л Ҳмһ„ нҳём¶ңлҗҳлҠ” л ҢлҚ”л§Ғ лЈЁн”„ лӮҙл¶Җм—җм„ң м „мІҙ мә”лІ„мҠӨлҘј мҙҲкё°нҷ”н•ҳкі , нҠ№м • н”Ҫм…Җмқҳ мң„м№ҳлҘј ліҖкІҪн•ҳлҠ” н•өмӢ¬ м•Ңкі лҰ¬мҰҳмқҖ лӢӨмқҢкіј к°ҷмқҙ м¶”мғҒнҷ”н•  мҲҳ мһҲмҠөлӢҲлӢӨ.

    Function UpdateFrame():
    // 1. мғҒнғң мң м§Җ ліҖмҲҳ (м •м Ғ ліҖмҲҳлЎң н”„л Ҳмһ„к°„ мғҒнғң кіөмң )
    Static Variable current_pixel_index = 0

    // 2. мә”лІ„мҠӨ лІ„нҚј мҙҲкё°нҷ” (лӘЁл“  н”Ҫм…Җмқ„ нҡҢмғүмңјлЎң мұ„мӣҖ)
    Define Canvas = Array of size (Width * Height) with Color(0.8, 0.8, 0.8)

    // 3. нҠ№м • мң„м№ҳмқҳ н”Ҫм…Җ мғүмғҒмқ„ л№Ёк°„мғүмңјлЎң ліҖкІҪ
    Canvas[current_pixel_index] = Color(1.0, 0.0, 0.0)

    // 4. лӢӨмқҢ н”„л Ҳмһ„мқ„ мң„н•ң мқёлҚұмҠӨ мҰқк°Җ л°Ҹ кІҪкі„ кІҖмӮ¬ (л°©м–ҙм Ғ мҙҲкё°нҷ”)
    current_pixel_index = current_pixel_index + 1
    If current_pixel_index >= Canvas.Length - 1 Then
        current_pixel_index = 0
    EndIf

    // 5. ліҖкІҪлҗң лІ„нҚјлҘј к·ёлһҳн”Ҫ м№ҙл“ң(VRAM)м—җ м—…лЎңл“ң л°Ҹ нҷ”л©ҙ м¶ңл Ҙ
    UploadToGPU(Canvas)

    3. WebGPU мқён„°лһҷнӢ°лёҢ лҚ°лӘЁ

    мқҙ к°ңл…җмқ„ мӣ№ лёҢлқјмҡ°м Җ н‘ңмӨҖ м ҖмҲҳмӨҖ к·ёлһҳн”ҪмҠӨ APIмқё WebGPU нҷҳкІҪм—җм„ң кө¬нҳ„н•ңлӢӨл©ҙ, CPUк°Җ м•„лӢҢ GPUмқҳ лі‘л ¬ м»ҙн“ЁнҠё м…°мқҙлҚ”(Compute Shader) нҳ№мқҖ н”„лһҳк·ёлЁјнҠё м…°мқҙлҚ”(Fragment Shader)лҘј нҶөн•ҙ нӣЁм”¬ лҚ” нҡЁмңЁм ҒмңјлЎң мӢңк°Ғнҷ”н•  мҲҳ мһҲмҠөлӢҲлӢӨ.

    // WebGPU WGSL (WebGPU Shading Language) мҳҲмӢң м»Ём…ү

struct Uniforms {
    canvas_size : vec2<f32>,
    frame_index : u32,
}
@group(0) @binding(0) var<uniform> ubo : Uniforms;

@fragment
fn fs_main(@builtin(position) frag_coord : vec4<f32>) -> @location(0) vec4<f32> {
    // 2м°Ёмӣҗ н”„лһҳк·ёлЁјнҠё мўҢн‘ңлҘј 1м°Ёмӣҗ мқёлҚұмҠӨлЎң ліҖнҷҳ
    let x = u32(frag_coord.x);
    let y = u32(frag_coord.y);
    let width = u32(ubo.canvas_size.x);
    let current_index = x + (y * width);

    // CPUм—җм„ң л§Ө н”„л Ҳмһ„ мҰқк°Җн•ҳм—¬ м „лӢ¬лҗң frame_indexмҷҖ мқјм№ҳн•ҳлҠ” н”Ҫм…Җл§Ң л№Ёк°„мғүмңјлЎң к·ёлҰј
    let target_index = ubo.frame_index % (u32(ubo.canvas_size.x * ubo.canvas_size.y));

    if (current_index == target_index) {
        return vec4<f32>(1.0, 0.0, 0.0, 1.0); // Red
    }
    return vec4<f32>(0.8, 0.8, 0.8, 1.0); // Background Gray
}

    лёҢлқјмҡ°м Җм—җм„ңмқҳ мӢңк°Ғнҷ” лӘЁмҠө

    1. м •м Ғмқё л°°кІҪ мҶҚ мӣҖм§Ғмһ„: мӣ№ нҺҳмқҙм§Җ лӮҙ мғқм„ұлҗң мә”лІ„мҠӨ(Canvas) мң„м—җ мҲҳл§Ң к°ңмқҳ нҡҢмғү н”Ҫм…Җл“Өмқҙ кІ©мһҗ лӘЁм–‘мңјлЎң л°°м№ҳлҗ©лӢҲлӢӨ.
    2. кі мҶҚ лһҳмҠӨн„° мҠӨмә”: CPUк°Җ л§Ө н”„л Ҳмһ„ мҰқк°ҖмӢңнӮӨлҠ” frame_indexлҘј м „лӢ¬л°ӣм•„, лӢЁ н•ҳлӮҳмқҳ л№Ёк°„мғү н”Ҫм…Җмқҙ л§Ҳм№ҳ лёҢлқјмҡҙкҙҖ TVмқҳ м „мһҗмҙқмқҙ нҷ”л©ҙмқ„ нӣ‘кі  м§ҖлӮҳк°Җл“Ҝ мўҢмёЎ мғҒлӢЁм—җм„ң мҡ°мёЎ н•ҳлӢЁ л°©н–ҘмңјлЎң л№ лҘҙкІҢ нқҳлҹ¬к°‘лӢҲлӢӨ.
    3. л¬ҙн•ң лЈЁн”„: л§Ҳм§Җл§ү н”Ҫм…Җм—җ лҸ„лӢ¬н•ҳлҠ” мҲңк°„, м§Җм—° м—Ҷмқҙ лӢӨмӢң мІ« лІҲм§ё н”Ҫм…Җ(мўҢмёЎ мғҒлӢЁ 0лІҲ)лЎң лҸҢм•„мҷҖ м• лӢҲл©”мқҙм…ҳмқҙ л¶Җл“ңлҹҪкІҢ л°ҳліөлҗ©лӢҲлӢӨ.
