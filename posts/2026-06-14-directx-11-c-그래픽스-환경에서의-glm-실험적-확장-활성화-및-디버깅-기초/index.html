    [DirectX 11] C++ к·ёлһҳн”ҪмҠӨ нҷҳкІҪм—җм„ңмқҳ GLM мӢӨн—ҳм Ғ нҷ•мһҘ нҷңм„ұнҷ” л°Ҹ л””лІ„к№… кё°мҙҲ

    мқҙлІҲ м»Өл°ӢмқҖ DirectX 11 / C++ к·ёлһҳн”ҪмҠӨ н•ҷмҠө кіјм •м—җм„ң мҲҳн•ҷ лқјмқҙлёҢлҹ¬лҰ¬мқё GLM(OpenGL Mathematics)мқҳ л””лІ„к№… мң нӢёлҰ¬нӢ°лҘј м•Ҳм „н•ҳкІҢ нҶөн•©н•ҳкё° мң„н•ң н”„лЎңм қнҠё м„Өм • ліҖкІҪмқ„ лӢҙкі  мһҲмҠөлӢҲлӢӨ.

    лӢЁмҲңнһҲ л№Ңл“ң м—җлҹ¬лҘј н•ҙкІ°н•ң кІғмқ„ л„ҳм–ҙ, к·ёлһҳн”ҪмҠӨ н”„лЎңк·ёлһҳл°Қм—җм„ң н•„мҲҳм Ғмқё мҲҳн•ҷм Ғ лҚ°мқҙн„° мӢңк°Ғнҷ”(л””лІ„к№…)мқҳ кё°нӢҖмқ„ л§Ҳл Ён–ҲлӢӨлҠ” м җм—җм„ң мқҳлҜёк°Җ нҒҪлӢҲлӢӨ.

    1. м»Өл°Ӣмқҳ н•өмӢ¬ к°ңл…җ: к·ёлһҳн”ҪмҠӨ л””лІ„к№…кіј GLM л§ӨнҒ¬лЎң м„Өм •

    3D ліҖнҷҳ MatrixмҷҖ л””лІ„к№…мқҳ н•ңкі„

    к·ёлһҳн”ҪмҠӨ нҢҢмқҙн”„лқјмқёмқ„ кө¬м¶•н•  л•Ң, мҡ°лҰ¬лҠ” мҲҳл§ҺмқҖ лІЎн„°мҷҖ н–үл ¬ м—°мӮ°мқ„ мҲҳн–үн•©лӢҲлӢӨ. лЎңм»¬ мўҢн‘ңкі„мқҳ м •м җмқ„ мҠӨнҒ¬лҰ° кіөк°„мңјлЎң ліҖнҷҳн•ҳлҠ” лҢҖн‘ңм Ғмқё кіөмӢқмқҖ лӢӨмқҢкіј к°ҷмҠөлӢҲлӢӨ.

    \[ \mathbf{v}_{clip} = P \times V \times M \times \mathbf{v}_{local} \]

    м—¬кё°м„ң \(M\)(Model), \(V\)(View), \(P\)(Projection)лҠ” лӘЁл‘җ \(4 \times 4\) н–үл ¬мһ…лӢҲлӢӨ. м…°мқҙлҚ”лӮҳ C++ м• н”ҢлҰ¬мјҖмқҙм…ҳ лӢЁкі„м—җм„ң ліҖнҷҳмқҙ мһҳлӘ»лҗҳм—Ҳмқ„ л•Ң, мқҙлҘј мҪҳмҶ”м—җ м¶ңл Ҙн•ҳм—¬ к°’мқ„ нҷ•мқён•ҳлҠ” кІғмқҖ л””лІ„к№…мқҳ м¶ңл°ңм җмһ…лӢҲлӢӨ.

    н•ҳм§Җл§Ң C++ н‘ңмӨҖ лқјмқҙлёҢлҹ¬лҰ¬(std::cout)лҠ” glm::vec3лӮҳ glm::mat4 к°ҷмқҖ кі м°Ёмӣҗ мҲҳн•ҷ нғҖмһ…мқ„ кё°ліём ҒмңјлЎң м¶ңл Ҙн•  мҲҳ м—ҶмҠөлӢҲлӢӨ.

    glm/gtx/string_cast.hppмҷҖ мӢӨн—ҳм Ғ нҷ•мһҘ кё°лҠҘ

    GLM лқјмқҙлёҢлҹ¬лҰ¬лҠ” мқҙлҘј н•ҙкІ°н•ҳкё° мң„н•ҙ н–үл ¬кіј лІЎн„°лҘј л¬ёмһҗм—ҙлЎң мүҪкІҢ ліҖнҷҳн•ҙ мЈјлҠ” glm::to_string() н•ЁмҲҳлҘј м ңкіөн•©лӢҲлӢӨ. мқҙ н•ЁмҲҳлҠ” glm/gtx/string_cast.hpp н—ӨлҚ”м—җ м •мқҳлҗҳм–ҙ мһҲмҠөлӢҲлӢӨ.

    • GTX(Extension)мқҳ мқҳлҜё: GLMм—җм„ң gtx кІҪлЎңм—җ мһҲлҠ” н—ӨлҚ”л“ӨмқҖ 'мӢӨн—ҳм Ғ(Experimental)' кё°лҠҘмқ„ мқҳлҜён•©лӢҲлӢӨ. н–Ҙнӣ„ APIк°Җ ліҖкІҪлҗ  к°ҖлҠҘм„ұмқҙ мһҲмңјлҜҖлЎң, к°ңл°ңмһҗк°Җ мқҙ мң„н—ҳмқ„ мқём§Җн•ҳкі  мӮ¬мҡ©н•ҳкІ лӢӨлҠ” лӘ…мӢңм Ғмқё лҸҷмқҳлҘј мҡ”кө¬н•©лӢҲлӢӨ.
    • н•ҙкІ°мұ…: н”„лҰ¬н”„лЎңм„ём„ң(м „мІҳлҰ¬кё°) лӢЁкі„м—җм„ң GLM_ENABLE_EXPERIMENTAL л§ӨнҒ¬лЎңлҘј м •мқҳн•ҙм•јл§Ң н•ҙлӢ№ н—ӨлҚ”мқҳ кё°лҠҘмқ„ м»ҙнҢҢмқјлҹ¬к°Җ н—Ҳмҡ©н•©лӢҲлӢӨ. н•ҷмҠөмһҗлҠ” мқҙлҘј н”„лЎңм қнҠё м „л°ҳ(.vcxproj м„Өм •)м—җ л°ҳмҳҒн•ҳм—¬ н•ҙкІ°н–ҲмҠөлӢҲлӢӨ.

    2. н•ҷмҠөмһҗк°Җ м§Ғм ‘ кө¬нҳ„н•ҳл©° мқҙн•ҙн•ң н•өмӢ¬ кө¬мЎ°

    н•ҷмҠөмһҗлҠ” лӢЁмҲң мҶҢмҠӨ мҪ”л“ң л ҲлІЁ(#define)м—җм„ң л§ӨнҒ¬лЎңлҘј м„ м–ён•ҳлҠ” лҢҖмӢ , Visual Studio н”„лЎңм қнҠё кө¬м„ұ мҡ”мҶҢ(.vcxproj)м—җ м§Ғм ‘ м •мқҳлҘј м¶”к°Җн–ҲмҠөлӢҲлӢӨ. мқҙлҠ” к·ңлӘЁк°Җ нҒ° к·ёлһҳн”ҪмҠӨ м—”м§„ м•„нӮӨн…ҚмІҳм—җм„ң л§Өмҡ° к¶ҢмһҘлҗҳлҠ” л°©мӢқмһ…лӢҲлӢӨ.

    м „мІҳлҰ¬ м •мқҳ нҶөн•©мқҳ м•„нӮӨн…ҚмІҳм Ғ мқҳлҜё

    лӘЁл“  л№Ңл“ң нҷҳкІҪ(Debug, Release, Win32, x64)м—җ л§ӨнҒ¬лЎңлҘј мЈјмһ…н•ЁмңјлЎңмҚЁ лӢӨмқҢкіј к°ҷмқҖ мқҙм җмқ„ м–»мҠөлӢҲлӢӨ. 1. мқјкҙҖм„ұ: нҠ№м • мҶҢмҠӨ нҢҢмқјм—җм„ң #defineмқ„ лҲ„лқҪн•ҳм—¬ л°ңмғқн•ҳлҠ” мӮ°л°ңм Ғмқё л№Ңл“ң м—җлҹ¬лҘј л°©м§Җн•©лӢҲлӢӨ. 2. мӮ¬м „ м»ҙнҢҢмқјлҗң н—ӨлҚ”(PCH)мҷҖмқҳ нҳёнҷҳм„ұ: к·ёлһҳн”ҪмҠӨ н”„лЎңм қнҠём—җм„ң нқ”нһҲ м“°мқҙлҠ” pch.h л“ұкіј м—үнӮӨм§Җ м•Ҡкі  м•Ҳм „н•ҳкІҢ GLM м„Өм •мқ„ м „м—ӯм—җ м „нҢҢн•©лӢҲлӢӨ.

    м¶”мғҒнҷ”лҗң л””лІ„к№… мӣҢнҒ¬н”ҢлЎңмҡ°

    н•ҷмҠөмһҗк°Җ кө¬м¶•н•ң л””лІ„к№… нҷҳкІҪмқҳ л…јлҰ¬м Ғ нқҗлҰ„мқҖ лӢӨмқҢкіј к°ҷмқҙ мҡ”м•Ҫн•  мҲҳ мһҲмҠөлӢҲлӢӨ.

    // 1. к·ёлһҳн”ҪмҠӨ л ҢлҚ”л§Ғ лЈЁн”„ мӨ‘ м№ҙл©”лқј лҳҗлҠ” мҳӨлёҢм қнҠёмқҳ н–үл ¬ кі„мӮ°
Matrix4x4 modelMatrix = CalculateModelTransform(translation, rotation, scale);
Matrix4x4 viewMatrix  = CalculateViewMatrix(cameraPosition, lookAt);
Matrix4x4 projectionMatrix = CalculateProjectionMatrix(fov, aspectRatio, nearZ, farZ);

// 2. MVP кІ°н•© н–үл ¬ лҸ„м¶ң
Matrix4x4 mvpMatrix = projectionMatrix * viewMatrix * modelMatrix;

// 3. л№Ңл“ң м—җлҹ¬ м—Ҷмқҙ мһ‘лҸҷн•ҳлҠ” GLM л¬ёмһҗм—ҙ ліҖнҷҳ л””лІ„к№… (н•ҷмҠөмһҗк°Җ н•ҙкІ°н•ң н•өмӢ¬ мҳҒм—ӯ)
#ifdef _DEBUG
    // мҪҳмҶ” м°Ҫм—җ 4x4 н–үл ¬мқҳ мӣҗмҶҢл“Өмқ„ нҸ¬л§·нҢ…н•ҳм—¬ мӢӨмӢңк°„ м¶ңл Ҙ
    String formattedMatrix = Math::ToString(mvpMatrix);
    Debug::Log("Current MVP Matrix:\n" + formattedMatrix);
#endif

    3. WebGPU мқён„°лһҷнӢ°лёҢ лҚ°лӘЁлЎңмқҳ нҷ•мһҘ кҙҖм җ

    мӣ№ нҷҳкІҪмқҳ м°Ём„ёлҢҖ к·ёлһҳн”ҪмҠӨ APIмқё WebGPUм—җм„ңлҸ„ мқҙмҷҖ мң мӮ¬н•ң мҲҳн•ҷм Ғ л””лІ„к№… л°Ҹ м—°мӮ° л¬ём ңк°Җ л°ңмғқн•©лӢҲлӢӨ. WebGPUмқҳ м…°мқҙлҚ” м–ём–ҙмқё WGSL(WebGPU Shading Language) м—ӯмӢң ліөмһЎн•ң н–үл ¬ м—°мӮ°мқ„ мҲҳн–үн•ҳм§Җл§Ң, м…°мқҙлҚ” лӮҙл¶Җмқҳ к°’мқ„ CPU(JavaScript) мёЎм—җм„ң м§Ғм ‘ нҷ•мқён•ҳлҠ” кІғмқҖ л§Өмҡ° к№ҢлӢӨлЎӯмҠөлӢҲлӢӨ.

    мқҙ м»Өл°Ӣм—җм„ң лӢӨлЈ¬ "мҲҳн•ҷм Ғ м—°мӮ° кІ°кіјлҘј CPUлЎң к°Җм ёмҷҖ нҷ•мқён•ҳлҠ” мӢңк°Ғнҷ” кё°лІ•"мқ„ WebGPU Compute Shader лҚ°лӘЁлЎң мһ¬кө¬м„ұн•ңлӢӨл©ҙ лӢӨмқҢкіј к°ҷмқҖ л°©мӢқмңјлЎң мӢңк°Ғнҷ”н•  мҲҳ мһҲмҠөлӢҲлӢӨ.

    WebGPU Compute Shader кё°л°ҳ н–үл ¬ ліҖнҷҳ мӢңк°Ғнҷ” мӢңлӮҳлҰ¬мҳӨ

    1. мғҒнҷ© м„Өм •: мҲҳл§Ң к°ңмқҳ м •м җ(Vertices)м—җ мһ„мқҳмқҳ 3D нҡҢм „ н–үл ¬ \(R(\theta)\)лҘј м Ғмҡ©н•ҳлҠ” Compute ShaderлҘј к°ҖлҸҷн•©лӢҲлӢӨ.
    2. л””лІ„к№… лІ„нҚј(Storage Buffer) мғқм„ұ:
    3. CPUмҷҖ GPUк°Җ кіөмң н•ҳлҠ” readback лІ„нҚјлҘј л§Ңл“ӯлӢҲлӢӨ.
    4. GPU м—°мӮ° кІ°кіјлЎң лҸ„м¶ңлҗң ліҖнҷҳ н–үл ¬ л°Ҹ м •м җ мң„м№ҳ м •ліҙлҘј мқҙ лІ„нҚјм—җ кё°лЎқн•©лӢҲлӢӨ.
    5. JSлҘј нҶөн•ң мӢңк°Ғнҷ”(WebGPUмқҳ string_cast кө¬нҳ„):
    6. GPU м—°мӮ°мқҙ лҒқлӮҳл©ҙ GPUBuffer.mapAsync()лҘј нҳём¶ңн•ҳм—¬ лҚ°мқҙн„°лҘј л©”лӘЁлҰ¬м—җ л§Өн•‘н•©лӢҲлӢӨ.
    7. JavaScript мҪҳмҶ” лҳҗлҠ” HTML лёҢлқјмҡ°м Җ нҷ”л©ҙм—җ 3D ліҖнҷҳ н–үл ¬ мғҒнғңмҷҖ м •м җл“Өмқҳ ліҖнҷ”лҘј мӢӨмӢңк°„ н…ҚмҠӨнҠё л°Ҹ 3D кё°н•ҳкө¬мЎ°лЎң л ҢлҚ”л§Ғн•©лӢҲлӢӨ.
    // WGSL (Compute Shader мҳҲмӢң)
struct DebugData {
    transformMatrix : mat4x4<f32>,
    calculatedPos   : vec4<f32>,
}
@group(0) @binding(0) var<storage, read_write> debugBuffer : DebugData;

@compute @workgroup_size(1)
fn main() {
    // GPU лӮҙл¶Җм—җм„ң ліөмһЎн•ң н–үл ¬ м—°мӮ° мҲҳн–ү
    let M = mat4x4<f32>(...);
    debugBuffer.transformMatrix = M; 
    debugBuffer.calculatedPos = M * vec4<f32>(1.0, 2.0, 3.0, 1.0);
}
