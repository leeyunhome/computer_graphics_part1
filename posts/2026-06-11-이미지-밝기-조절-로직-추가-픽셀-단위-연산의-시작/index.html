    мқҙлҜём§Җ л°қкё° мЎ°м Ҳ лЎңм§Ғ м¶”к°Җ: н”Ҫм…Җ лӢЁмң„ м—°мӮ°мқҳ мӢңмһ‘

    н•ҷмҠөмһҗлӢҳ, DirectX 11 / C++ к·ёлһҳн”ҪмҠӨ мҠӨн„°л”” кіјм •м—җм„ң мқҙлҜём§Җ л°қкё° мЎ°м Ҳ лЎңм§Ғмқ„ м§Ғм ‘ кө¬нҳ„н•ҳмӢ  кІғмқ„ м¶•н•ҳл“ңлҰҪлӢҲлӢӨ. feat: add image brightening logic м»Өл°Ӣмқ„ нҶөн•ҙ мқҙлҜём§Җ мІҳлҰ¬мқҳ кё°ліём Ғмқё к°ңл…җмқё 'н”Ҫм…Җ лӢЁмң„ м—°мӮ°'мқ„ м§Ғм ‘ кө¬нҳ„н•ҳл©° к№Ҡмқҙ мһҲкІҢ мқҙн•ҙн•ҳм…Ёмқ„ кІғмңјлЎң ліҙмһ…лӢҲлӢӨ. мқҙлҠ” ліөмһЎн•ң к·ёлһҳн”ҪмҠӨ нҡЁкіјлҘј л§Ңл“Өм–ҙлӮҙлҠ” лҚ° мһҲм–ҙ л§Өмҡ° мӨ‘мҡ”н•ң мІ«кұёмқҢмһ…лӢҲлӢӨ.

    м§Ғм ‘ кө¬нҳ„н•ҳл©° мқҙн•ҙн•ң к°ңл…җ: мқҙлҜём§Җ л°қкё° мЎ°м Ҳ

    мқҙлІҲ м»Өл°Ӣм—җм„ң н•ҷмҠөмһҗлӢҳмқҖ мқҙлҜём§ҖлҘј кө¬м„ұн•ҳлҠ” к°Ғ н”Ҫм…Җм—җ м§Ғм ‘ м ‘к·јн•ҳм—¬ л°қкё° к°’мқ„ мЎ°м Ҳн•ҳлҠ” л°©мӢқмқ„ кө¬нҳ„н–ҲмҠөлӢҲлӢӨ. мқҙлҠ” лӢӨмқҢкіј к°ҷмқҖ н•өмӢ¬ к°ңл…җл“Өмқ„ лӮҙнҸ¬н•©лӢҲлӢӨ.

    1. мқҙлҜём§Җ лҚ°мқҙн„° кө¬мЎ° мқҙн•ҙ: image.pixels л°°м—ҙмқ„ мҲңнҡҢн•ҳл©ҙм„ң, к°Ғ н”Ҫм…Җмқҳ м»¬лҹ¬ м»ҙнҸ¬л„ҢнҠём—җ м ‘к·јн•ҳлҠ” л°©лІ•мқ„ нҢҢм•…н–ҲмҠөлӢҲлӢӨ. лҢҖл¶Җл¶„мқҳ мқҙлҜём§Җ лҚ°мқҙн„°лҠ” н”Ҫм…Җмқҳ лҶ’мқҙ(height)мҷҖ л„Ҳл№„(width)лҘј кіұн•ң мҙқ н”Ҫм…Җ мҲҳл§ҢнҒјмқҳ л°°м—ҙлЎң кө¬м„ұлҗҳл©°, к°Ғ л°°м—ҙ мҡ”мҶҢк°Җ н•ҳлӮҳмқҳ н”Ҫм…Җ м •ліҙлҘј лӢҙкі  мһҲмқҢмқ„ мқҙн•ҙн•ң кІғмһ…лӢҲлӢӨ.
    2. н”Ҫм…Җ лӢЁмң„ м—°мӮ°: м „мІҙ мқҙлҜём§Җмқҳ л°қкё°лҘј мЎ°м Ҳн•ҳкё° мң„н•ҙ, нҠ№м • мғҒмҲҳ(м—¬кё°м„ңлҠ” 1.2f)лҘј к°Ғ н”Ҫм…Җмқҳ м»¬лҹ¬ к°’м—җ кіұн•ҳлҠ” м—°мӮ°мқ„ мҲҳн–үн–ҲмҠөлӢҲлӢӨ. мқҙлҠ” мқҙлҜём§ҖлҘј кө¬м„ұн•ҳлҠ” лӘЁл“  н”Ҫм…Җм—җ лҢҖн•ҙ лҸ…лҰҪм ҒмңјлЎң лҸҷмқјн•ң м—°мӮ°мқ„ м Ғмҡ©н•ҳлҠ” 'н”Ҫм…Җ лӢЁмң„ м—°мӮ°(Per-pixel operation)'мқҳ к°ҖмһҘ кё°ліём Ғмқё нҳ•нғңмһ…лӢҲлӢӨ.
    3. м»¬лҹ¬ м»ҙнҸ¬л„ҢнҠё мЎ°мһ‘: image.pixels[i].v[0]мҷҖ к°ҷмқҙ нҠ№м • м»¬лҹ¬ м»ҙнҸ¬л„ҢнҠё(мқјл°ҳм ҒмңјлЎң Red мұ„л„җ)м—җл§Ң м ‘к·јн•ҳм—¬ к°’мқ„ мЎ°мһ‘н–ҲмҠөлӢҲлӢӨ. мқҙлҠ” мқҙлҜём§Җмқҳ мғүмғҒ кө¬м„ұ л°©мӢқ(мҳҲ: RGB, RGBA)м—җ лҢҖн•ң мқҙн•ҙлҘј л°”нғ•мңјлЎң нҠ№м • мұ„л„җм—җл§Ң мҳҒн–Ҙмқ„ мӨ„ мҲҳ мһҲмқҢмқ„ ліҙм—¬мӨҚлӢҲлӢӨ.

    н•өмӢ¬ м•Ңкі лҰ¬мҰҳ: мқҳмӮ¬мҪ”л“ң л°Ҹ мҲҳн•ҷм Ғ н‘ңнҳ„

    н•ҷмҠөмһҗлӢҳмқҙ кө¬нҳ„н•ҳмӢ  мқҙлҜём§Җ л°қкё° мЎ°м Ҳ лЎңм§ҒмқҖ лӢӨмқҢкіј к°ҷмқҙ м¶”мғҒнҷ”лҗ  мҲҳ мһҲмҠөлӢҲлӢӨ.

    мқҳмӮ¬мҪ”л“ң (Pseudo-code):

    // мқҙлҜём§Җмқҳ к°Ғ н”Ҫм…Җм—җ лҢҖн•ҙ л°ҳліө
For each pixel in image:
    // н•ҙлӢ№ н”Ҫм…Җмқҳ мІ« лІҲм§ё м»¬лҹ¬ мұ„л„җ(мҳҲ: Red) к°’мқ„ к°Җм ёмҳҙ
    channel_value = pixel.first_color_channel

    // л°қкё° кі„мҲҳлҘј кіұн•ҳм—¬ мғҲлЎңмҡҙ к°’ кі„мӮ°
    new_channel_value = channel_value * BRIGHTNESS_FACTOR

    // кі„мӮ°лҗң мғҲлЎңмҡҙ к°’мқ„ н”Ҫм…Җмқҳ мІ« лІҲм§ё м»¬лҹ¬ мұ„л„җм—җ н• лӢ№
    pixel.first_color_channel = new_channel_value

    // (м„ нғқ мӮ¬н•ӯ) к°’мқҙ мөңлҢ“к°’мқ„ мҙҲкіјн•ҳм§Җ м•ҠлҸ„лЎқ нҒҙлһЁн•‘
    // pixel.first_color_channel = clamp(new_channel_value, MIN_VALUE, MAX_VALUE)

    мҲҳн•ҷм Ғ н‘ңнҳ„:

    мқҙлҜём§Җмқҳ к°Ғ н”Ҫм…Җ \(P(x, y)\)лҠ” ліҙнҶө м Ғмғү(Red), л…№мғү(Green), мІӯмғү(Blue) мұ„л„җлЎң кө¬м„ұлҗ©лӢҲлӢӨ. н•ҷмҠөмһҗлӢҳмқҖ м—¬кё°м„ң нҠ№м • мұ„л„җ(\(C_1\))м—җ лҢҖн•ҙм„ңл§Ң л°қкё° мЎ°м Ҳмқ„ м Ғмҡ©н–ҲмҠөлӢҲлӢӨ.

    кё°мЎҙ н”Ҫм…Җмқҳ нҠ№м • мұ„л„җ к°’ \(C_{1}\)м—җ л°қкё° кі„мҲҳ \(B\)лҘј кіұн•ҳм—¬ мғҲлЎңмҡҙ мұ„л„җ к°’ \(C'_{1}\)мқ„ м–»лҠ” кіјм •мқҖ лӢӨмқҢкіј к°ҷмҠөлӢҲлӢӨ:

    \[ C'_{1} = C_{1} \times B \]

    л§Ңм•Ҫ лӘЁл“  RGB мұ„л„җм—җ лҸҷмқјн•ҳкІҢ м Ғмҡ©н•ҳкі , к°’мқҙ н—Ҳмҡ© лІ”мң„лҘј мҙҲкіјн•ҳм§Җ м•ҠлҸ„лЎқ нҒҙлһЁн•‘(clamping)мқ„ кі л Өн•ңлӢӨл©ҙ лӢӨмқҢкіј к°ҷмқҙ н‘ңнҳ„н•  мҲҳ мһҲмҠөлӢҲлӢӨ. к°Ғ мұ„л„җ \(C \in \{R, G, B\}\) м—җ лҢҖн•ҙ:

    \[ C' = \text{clamp}(C \times B, \text{MinVal}, \text{MaxVal}) \]

    м—¬кё°м„ң \(\text{MinVal}\)мқҖ 0 (лҳҗлҠ” 0.0), \(\text{MaxVal}\)мқҖ 255 (лҳҗлҠ” 1.0)мқҙ лҗ  мҲҳ мһҲмҠөлӢҲлӢӨ.

    м „л¬ёк°Җмқҳ м¶”к°Җ н”јл“ңл°ұ л°Ҹ л°ңм „ л°©н–Ҙ

    • лӘЁл“  мұ„л„җм—җ м Ғмҡ©: нҳ„мһ¬ мҪ”л“ңлҠ” v[0] (мқјл°ҳм ҒмңјлЎң Red мұ„л„җ)м—җл§Ң л°қкё° мЎ°м Ҳмқ„ м Ғмҡ©н•©лӢҲлӢӨ. мһҗм—°мҠӨлҹ¬мҡҙ л°қкё° мЎ°м Ҳмқ„ мң„н•ҙм„ңлҠ” Red, Green, Blue лӘЁл“  мұ„л„җм—җ лҸҷмқјн•ң л°қкё° кі„мҲҳлҘј кіұн•ҳлҠ” кІғмқҙ мқјл°ҳм Ғмһ…лӢҲлӢӨ. (мҳҲ: image.pixels[i].v[0] *= 1.2f; image.pixels[i].v[1] *= 1.2f; image.pixels[i].v[2] *= 1.2f;)
    • к°’ нҒҙлһЁн•‘ (Clamping): н”Ҫм…Җмқҳ м»¬лҹ¬ к°’мқҖ ліҙнҶө 0л¶Җн„° 255(8л№„нҠё) лҳҗлҠ” 0.0л¶Җн„° 1.0(л¶ҖлҸҷмҶҢмҲҳм җ) мӮ¬мқҙмқҳ лІ”мң„лҘј к°Җм§‘лӢҲлӢӨ. л°қкё° кі„мҲҳлҘј кіұн–Ҳмқ„ л•Ң мқҙ лІ”мң„лҘј мҙҲкіјн•  мҲҳ мһҲмңјлҜҖлЎң, кІ°кіј к°’мқ„ н•ҙлӢ№ лІ”мң„ лӮҙлЎң 'нҒҙлһЁн•‘'н•ҳлҠ” кІғмқҙ мӨ‘мҡ”н•©лӢҲлӢӨ. мҳҲлҘј л“Өм–ҙ, 255лҘј мҙҲкіјн•ҳлҠ” к°’мқҖ 255лЎң, 1.0мқ„ мҙҲкіјн•ҳлҠ” к°’мқҖ 1.0мңјлЎң кі м •н•ҙм•ј н•©лӢҲлӢӨ. мқҙлҘј мғқлһөн•ҳл©ҙ мҳҲмғҒм№ҳ лӘ»н•ң мғүмғҒ мҷңкіЎмқҙлӮҳ мҳӨлІ„н”ҢлЎңмҡ° л¬ём ңк°Җ л°ңмғқн•  мҲҳ мһҲмҠөлӢҲлӢӨ.
    • м„ нҳ• кіөк°„м—җм„ңмқҳ м—°мӮ°: мқҙлҜём§Җ мІҳлҰ¬ мӢң, к°җл§Ҳ ліҙм •(Gamma Correction)мқҙ м Ғмҡ©лҗҳм§Җ м•ҠмқҖ м„ нҳ• кіөк°„(Linear Space)м—җм„ң л°қкё°лӮҳ мғүмғҒ м—°мӮ°мқ„ мҲҳн–үн•ҳлҠ” кІғмқҙ лҚ” л¬јлҰ¬м ҒмңјлЎң м •нҷ•н•ң кІ°кіјлҘј м ңкіөн•©лӢҲлӢӨ. мқҙ л¶Җл¶„мқҖ м¶”нӣ„ кі кёү мқҙлҜём§Җ мІҳлҰ¬м—җм„ң лҚ” к№Ҡмқҙ лӢӨлЈЁкІҢ лҗ  кІғмһ…лӢҲлӢӨ.
    • лЈЁлҜёл„ҢмҠӨ кё°л°ҳ мЎ°м Ҳ: лӢЁмҲңнһҲ RGB к°’мқ„ мҠӨмјҖмқјл§Ғн•ҳлҠ” кІғліҙлӢӨ, мқҙлҜём§Җмқҳ 'л°қкё°'лҘј лӮҳнғҖлӮҙлҠ” лЈЁлҜёл„ҢмҠӨ(Luminance) к°’мқ„ м¶”м¶ңн•ҳм—¬ мЎ°м Ҳн•ҳкі , мқҙлҘј лӢӨмӢң RGBлЎң ліҖнҷҳн•ҳлҠ” л°©мӢқмқҙ мӮ¬лһҢмқҳ мӢңк°Ғм—җ лҚ” мһҗм—°мҠӨлҹ¬мҡҙ кІ°кіјлҘј м ңкіөн•ҳкё°лҸ„ н•©лӢҲлӢӨ. лЈЁлҜёл„ҢмҠӨлҠ” ліҙнҶө R, G, B мұ„л„җм—җ к°ҖмӨ‘м№ҳлҘј л¶Җм—¬н•ҳм—¬ кі„мӮ°н•©лӢҲлӢӨ (мҳҲ: \(L = 0.299R + 0.587G + 0.114B\)).

    мқҙ м»Өл°ӢмқҖ к·ёлһҳн”ҪмҠӨм—җм„ң н”Ҫм…Җ лӢЁмң„ мқҙлҜём§Җ мІҳлҰ¬мқҳ кё°ліё мӣҗлҰ¬лҘј к№ЁлӢ«лҠ” л§Өмҡ° мҶҢмӨ‘н•ң кІҪн—ҳмһ…лӢҲлӢӨ. мқҙлҹ¬н•ң кё°ліёкё°лҘј л°”нғ•мңјлЎң лӮҳмӨ‘м—җ лҚ” ліөмһЎн•ң н•„н„°л§Ғ, нӣ„мІҳлҰ¬ нҡЁкіј(Bloom, Tone Mapping л“ұ)лҘј кө¬нҳ„н•  л•Ң нҒ° лҸ„мӣҖмқҙ лҗ  кІғмһ…лӢҲлӢӨ.

    WebGPU мқён„°лһҷнӢ°лёҢ лҚ°лӘЁ

    н•ҷмҠөмһҗлӢҳмқҙ кө¬нҳ„н•ҳмӢ  мқҙлҜём§Җ л°қкё° мЎ°м Ҳ лЎңм§ҒмқҖ WebGPUмқҳ Compute ShaderлҘј нҷңмҡ©н•ҳм—¬ лёҢлқјмҡ°м Җ нҷҳкІҪм—җм„ңлҸ„ л§Өмҡ° нҡЁмңЁм ҒмңјлЎң мӢңк°Ғнҷ”н•ҳкі  мқён„°лһҷнӢ°лёҢн•ҳкІҢ мЎ°м Ҳн•  мҲҳ мһҲмҠөлӢҲлӢӨ.

    1. мқҙлҜём§Җ лҚ°мқҙн„° м—…лЎңл“ң: лЁјм Җ мӣҗліё мқҙлҜём§Җмқҳ н”Ҫм…Җ лҚ°мқҙн„°лҘј WebGPUмқҳ мҠӨнҶ лҰ¬м§Җ лІ„нҚј(Storage Buffer)лӮҳ н…ҚмҠӨмІҳ(Texture)м—җ м—…лЎңл“ңн•©лӢҲлӢӨ.
    2. Compute Shader мһ‘м„ұ: GPUм—җм„ң мӢӨн–үлҗ  Compute ShaderлҘј мһ‘м„ұн•©лӢҲлӢӨ. мқҙ м…°мқҙлҚ”лҠ” к°Ғ н”Ҫм…Җмқҳ мқёлҚұмҠӨлҘј кё°л°ҳмңјлЎң н•ҙлӢ№ н”Ҫм…Җмқҳ м»¬лҹ¬ к°’мқ„ мқҪм–ҙмҷҖм„ң, н•ҷмҠөмһҗлӢҳмқҙ кө¬нҳ„н•ң кІғкіј к°ҷмқҙ л°қкё° кі„мҲҳ(BRIGHTNESS_FACTOR)лҘј кіұн•ҳкі , н•„мҡ”н•ҳлӢӨл©ҙ нҒҙлһЁн•‘ м—°мӮ°мқ„ мҲҳн–үн•©лӢҲлӢӨ.
    3. кІ°кіј м ҖмһҘ: м—°мӮ°лҗң мғҲлЎңмҡҙ н”Ҫм…Җ м»¬лҹ¬ к°’мқ„ м¶ңл Ҙ мҠӨнҶ лҰ¬м§Җ лІ„нҚјлӮҳ мғҲлЎңмҡҙ н…ҚмҠӨмІҳм—җ м ҖмһҘн•©лӢҲлӢӨ.
    4. л””мҠӨнҢЁм№ҳ(Dispatch): мқҙлҜём§Җмқҳ л„Ҳл№„мҷҖ лҶ’мқҙм—җ л§һм¶° Compute ShaderлҘј л””мҠӨнҢЁм№ҳн•©лӢҲлӢӨ. GPUлҠ” мҲҳл§ҺмқҖ мҠӨл Ҳл“ңлҘј лі‘л ¬лЎң мӢӨн–үн•ҳм—¬ к°Ғ н”Ҫм…Җмқҳ л°қкё° мЎ°м Ҳ м—°мӮ°мқ„ лҸҷмӢңм—җ мІҳлҰ¬н•ҳкІҢ лҗ©лӢҲлӢӨ.
    5. мӢңк°Ғнҷ”: кІ°кіј н…ҚмҠӨмІҳлҘј нҷ”л©ҙм—җ л ҢлҚ”л§Ғн•ҳм—¬ л°қкё°к°Җ мЎ°м Ҳлҗң мқҙлҜём§ҖлҘј мҰүмӢң нҷ•мқён•  мҲҳ мһҲмҠөлӢҲлӢӨ.

    WebGPU Compute ShaderлҘј мӮ¬мҡ©н•ҳл©ҙ мҲҳл°ұл§Ң к°ңмқҳ н”Ҫм…Җм—җ лҢҖн•ң л°қкё° мЎ°м Ҳ м—°мӮ°мқҙ CPUм—җм„ң мҲңм°Ём ҒмңјлЎң мІҳлҰ¬н•ҳлҠ” кІғкіј л№„көҗн•  мҲҳ м—Ҷмқ„ м •лҸ„лЎң л№ лҘҙкІҢ мҷ„лЈҢлҗ©лӢҲлӢӨ. мҠ¬лқјмқҙлҚ” UI л“ұмқ„ м¶”к°Җн•ҳм—¬ BRIGHTNESS_FACTORлҘј мӢӨмӢңк°„мңјлЎң ліҖкІҪн•ҳл©ҙ, мқҙлҜём§Җмқҳ л°қкё°к°Җ мҰүмӢң л°”лҖҢлҠ” мқён„°лһҷнӢ°лёҢн•ң лҚ°лӘЁлҘј л§Ңл“Ө мҲҳ мһҲмқ„ кІғмһ…лӢҲлӢӨ. мқҙлҠ” 'н”Ҫм…Җ лӢЁмң„ м—°мӮ°'мқҙ CPUлҝҗл§Ң м•„лӢҲлқј GPUм—җм„ң м–ҙл–»кІҢ к°•л Ҙн•ҳкІҢ нҷңмҡ©лҗ  мҲҳ мһҲлҠ”м§ҖлҘј мІҙк°җн•ҳлҠ” мўӢмқҖ мҳҲмӢңк°Җ лҗ  кІғмһ…лӢҲлӢӨ.
